歡迎使用 **即時電影資訊爬蟲**！這是一個專為電影愛好者與數據分析學習者設計的互動式網頁應用程式。

🔗 **線上體驗連結**: [點擊這裡前往應用程式](https://moviecrawler-erqpzfczqrkrkhsk94u3pn.streamlit.app/)

## 這是什麼？

這個網頁應用程式可以幫助您快速瀏覽並收集 [SSR1 Scrape Center](https://ssr1.scrape.center/) 網站上的電影資料。透過即時爬蟲技術，我們將分散在多個頁面的電影資訊整合在一個乾淨、美觀的介面中，讓您無需手動翻頁即可掌握所有資訊。

## 主要功能

*   **🚀 一鍵即時爬取**：只需點擊一個按鈕，程式便會自動掃描目標網站的 10 個頁面，抓取最新的電影資料。
*   **🃏 精美卡片展示**：以圖文並茂的卡片形式呈現每部電影的封面、名稱、評分與類型標籤。
*   **📊 自動統計分析**：爬取完成後，自動計算電影總數與平均評分，讓您對資料全貌一目了然。
*   **📥 資料匯出下載**：支援將爬取到的完整資料匯出為 CSV 檔案，方便您使用 Excel 或其他工具進行後續分析。
*   **📱 響應式設計**：無論是使用電腦、平板或手機，都能擁有流暢的瀏覽體驗。

## 如何使用？

1.  進入 [應用程式網頁](https://moviecrawler-erqpzfczqrkrkhsk94u3pn.streamlit.app/)。
2.  點擊左側選單的 **「🚀 開始爬取最新資料」** 按鈕。
3.  等待進度條跑完（約需幾秒鐘）。
4.  瀏覽右側生成的電影卡片列表。
5.  如需保存資料，點擊上方的 **「📥 下載 CSV 檔案」** 按鈕即可。
